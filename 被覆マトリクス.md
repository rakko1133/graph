# Excelグラフ編集機能 被覆マトリクス（根拠つき）

本書は「本アプリがExcelのグラフ編集機能をどれだけ満たしているか」を、**私の記憶ではなく Microsoft 公式の Excel オブジェクトモデル（列挙型・チャート要素）を母集合**として、1項目ずつ突き合わせた被覆（カバレッジ）マトリクスです。各行は「Excelの機能（公式出典）→ 実装状態 → アプリ側の根拠（ソース行/GUI）」で構成します。

凡例: ✅ 実装（完全）／△ 部分／✗ 未実装

---

## 1. 総合結果

**Excelのグラフ編集機能（公式オブジェクトモデル基準・81項目）のうち、本アプリの被覆は約 46%（37.5/81）。**

- 中核（作図の基本・系列書式・近似曲線・第2軸・凡例・タイトル・データ編集・サイズ/出力）は**高被覆**。
- 一方、**特殊グラフ種類・高度なエラーバー/データラベル/軸の書式・データテーブル・注釈図形・配色テーマ・3D**は**未実装**。
- → 以前の「Excelの全機能を実装」という表現は誤りで、正しくは「**主要機能を中心に約半分を実装**」です。

> 数え方の注意: グラフ種類は73の細分（3D/円錐/円柱/ピラミッド等の体裁違い）ではなく**ファミリ単位**で数えています。73値で数えると種類の被覆率はさらに下がり、総合も約30%台になります（後述）。

## 3. ギャップ一覧（未実装・部分の抜粋＝次に作るなら）

- **近似曲線**: 累乗(Power)、予測(Forecast)

- **軸**: 反転、表示単位、目盛間隔、数値書式、日付軸書式、軸の表示/非表示
- **目盛線**: 補助目盛線
- **系列書式**: グラデ/パターン塗り、枠線、平滑線
- **要素**: データテーブル、プロット/グラフエリア書式、注釈・図形、配色テーマ、上下バー/各種線

## 4. この被覆率の前提と限界（誠実な注記）

- **母集合の選び方に依存**: グラフ種類を「ファミリ」で数えたため種類被覆は43%。Excelの73値で数えると種類は約12/73≈16%となり、総合も約46%→約35%へ下がります。どちらも恣意ではなく、粒度の違いを明示しています。
- **対象はグラフ編集のみ**: Excel本体（数式・ピボット・条件付き書式・マクロ等）は対象外。"Excelの全機能" という比較自体は本来できません。
- **バージョン依存**: グラフ機能はExcel/Microsoft 365の版で増減します。出典は2024更新時点のVBAリファレンス。
- **OOXMLスキーマ基準だと更に細かい**: より厳密にはECMA-376のDrawingMLチャートスキーマ（`c:`名前空間）を母集合にでき、項目数は増えます（被覆率は下がる方向）。ご希望ならその基準でも作成します。
