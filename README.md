# CSV / TSV / 波形データ グラフ・解析ツール

CSV / TSV / 波形データを読み込み、グラフ描画・編集・解析ができる
デスクトップ GUI アプリです（**PySide6 / Qt** 製）。
日本語の列名・タイトル・軸ラベルに対応しています。

## 主な機能

- **複数ファイルのまとめ描画**: 複数の CSV/TSV/波形ファイルを読み込み、系列を重ねて比較
- **自動判定**: 文字コード（UTF-8 / BOM付き / Shift_JIS(CP932) など）と
  区切り文字（カンマ / タブ / セミコロン / パイプ）を自動判定。手動指定も可
- **8種のグラフ**: 折れ線 / 棒 / 横棒 / 積み上げ棒 / 散布図 / ヒストグラム / 箱ひげ / 円
- **Excel 相当のグラフ編集**: 系列ごとの色・線種・線幅・マーカー、軸範囲(min/max)、
  対数軸、凡例位置、タイトル/軸ラベルの文字サイズ
- **オシロスコープ表示**: time/div・V/div（**1-2-5プリセット＆`1ms`等の単位入力**）、X/Y 位置、
  div グリッド、全系列に合わせた自動スケール。
  **グラフを直接ドラッグして操作**（左ドラッグ=位置移動／右ドラッグ=time/V div／ホイール=time/div・
  Shift+ホイール=V/div、ドラッグ中は数値をオーバーレイ表示）
- **系列ON/OFFバー**: 描画グラフの上部に系列チェックを並べ、その場で表示を切替
- **カーソル測定**: クリックで2本設置し **Δt・ΔV・1/Δt** を読み取り。
  **カーソル線をドラッグで微調整**（波形に追従、Δはリアルタイム更新）
- **オシロ相当のデータ解析**（「4. 高度解析」タブ）:
  - **複数ピーク検出**（第1＝最大、第2、第3…と順位付け）。**平滑化**でノイズの多い実測データの偽ピークを抑制
  - **自動測定 約30種**：Vmax/Vmin/Vpp/Vmean/Vrms/σ、Top/Base、振幅、オーバー/アンダーシュート、
    周期、周波数（ゼロ交差・FFT）、立上り/立下り、+/-パルス幅、+/-デューティ、エッジ/サイクル数、
    **エッジ間時間（立上り→立上り/立下り→立下り/立上り→立下り/立下り→立上り）、Time@Max/Min、面積(∫)**
  - **サイクル統計**（周波数・Vppの min/max/平均/σ）と**トレンド表示**、**2chの位相差/遅延**
  - **数学チャンネル**：A±B / A×B / A÷B / 積分 / 微分 / 移動平均 / ローパス（結果を新波形として追加）
  - **FFT 詳細**：窓選択（hann/hamming/blackman/flattop/rect）、dB表示、**THD/SNR/SINAD/ENOB/SFDR**、
    **スペクトログラム**
  - **マスク/リミット合否テスト**、**アイダイアグラム**、**ジッタ解析（TIE・RMS・pp）**
  - **シリアルプロトコル解読**：UART / I2C / SPI
  - **表示範囲のみ測定**（ズーム/オシロ窓に追従）
- **大容量対応**: 50万点級でも **min/maxエンベロープ間引き**で高速表示し、ズーム時は自動で再サンプル。
  大きなファイルの読み込み・描画中は待機表示
- **設定の保存／読み込み**: JSON で設定を保存・復元。終了時に自動保存し、起動時に前回状態を復元
- **画像出力**: PNG / JPG / PDF / SVG / EPS で保存。**解像度(DPI)指定・背景透過**に対応し、
  ワンクリックで**クリップボードにコピー**も可能（拡大・パンの操作ツールも付属）

## 必要環境

- Python 3.12（推奨）／3.9 以上
- 依存ライブラリ: `pandas`, `matplotlib`, `PySide6`, `numpy`, `scipy`
- （任意）`spicelib` ＋ LTspice 本体 … LTspice 連携で波形を生成する場合のみ

専用の仮想環境 **`C:\.venv`** に依存をまとめる方式です（システムの Python に影響しません）。
初回は **`セットアップ.bat`** をダブルクリックすると、venv 作成と依存インストールを自動で行います。

```bat
:: セットアップ.bat が行う内容（手動で行う場合）
python -m venv C:\.venv
C:\.venv\Scripts\python -m pip install -r requirements.txt spicelib
```

> GUI は matplotlib の `qt_compat` 経由で実装しているため、`PySide6` の代わりに
> `PyQt6`（Qt6 系）でも同じコードで動作します。

## 起動方法

**`起動.bat`** をダブルクリック（`C:\.venv` の Python で起動。コンソール窓は出ません）。
うまく開かない場合はまず **`セットアップ.bat`** を実行してください。

コマンドから起動する場合:

```bat
C:\.venv\Scripts\pythonw graph_app.py
```

## 使い方の流れ

1. **データ**タブ →「ファイル追加」（**ドラッグ&ドロップ**可）で CSV/TSV を読み込む
2. **X軸**の列と、**Y軸（値）**の系列を選ぶ（複数可。**ダブルクリックで即描画**）
3. 「**グラフを描画**」(F5)。**「リアルタイム更新」がONなら設定変更が即反映**されます
4. **グラフ**タブで種別・タイトル・軸範囲・系列スタイルを編集（**系列名はダブルクリックで変更可**）
5. 波形なら **オシロ/解析**タブで「解析実行」→ ピーク・測定値、
   「オシロスコープ表示」で div 表示、「FFTスペクトル表示」で周波数解析
6. 設定は終了時に自動保存され、次回起動時に復元されます

> **ショートカット**: F5=描画 / Ctrl+O=ファイル追加 / Ctrl+S=画像保存 /
> Ctrl+Shift+C=クリップボードへコピー / Ctrl+R=解析 / F1=使い方。
> 主な機能は **解析・表示・ヘルプ メニュー**からも実行できます。軸範囲やtime/divの入力ミス
> （min≥max・数値でない・0以下）はステータスバーに警告が出ます。

### グラフ種別ごとの列の選び方

| 種別 | X軸 | Y軸 | 複数ファイル |
|------|-----|-----|------------|
| 折れ線 / 散布図 | 横軸の列 | 値の列を1つ以上 | ○（重ね描き） |
| 棒 / 横棒 / 積み上げ棒 | カテゴリの列 | 値の列を1つ以上 | 単一ファイル |
| ヒストグラム | （使わない） | 値の列を1つ以上 | ○ |
| 箱ひげ | （使わない） | 値の列を1つ以上 | ○ |
| 円 | ラベルの列 | 値の列を1つ | 単一ファイル |

## サンプルデータの生成

`サンプルデータ/` には動作確認用のデータが入っています。再生成・追加生成も可能です。

### 合成波形（numpy・LTspice不要）

```bash
python tools/generate_waveforms.py                       # 既定の一式を生成
python tools/generate_waveforms.py --type multitone --points 500000   # 点数指定
```

マルチトーン（FFTに複数ピーク）、減衰振動（時間波形に複数ピーク）、AM/チャープ/方形波、
負荷テスト用の大量データ（既定 50万点）などを生成します。

### 機能デモ用データ（各機能の性能を確認）

```bash
python tools/generate_showcase.py     # サンプルデータ/機能デモ/ に出力
```

各機能を実証するデータを生成します（数値検証済み）:
ノイズ込みピーク（平滑化）／高調波THD（≈5.4%）／ジッタ入りクロック（RMS≈3ns）／
**UART（"Hello, scope!"）・I2C・SPI** の波形／マスク試験用グリッチ／アイダイアグラム用NRZ。

### LTspice 連携（実回路の過渡解析）

```bash
python tools/generate_ltspice.py                         # 直列RLCの減衰リンギング
python tools/generate_ltspice.py --tstop 20m --maxstep 20n   # 点数を増やす
```

### ノイズ込み波形（LTspice・各種パターン）

```bash
python tools/generate_noisy_ltspice.py     # サンプルデータ/ノイズ波形/ に出力
```

LTspice の振る舞いソース＋乱数関数 `white()` で、ノイズ重畳の多彩な波形を実回路
シミュレーションで生成: 正弦(高/低SNR)・ノイズ&ジッタ入りクロック・RLC減衰振動＋ノイズ・
センサ風(ドリフト＋スパイク)・1/f近似/帯域制限の色付きノイズ・マルチトーン・AM・パルス列・三角波。

`spicelib` と LTspice 本体が必要です（本体は自動検出）。

### 電源・高速伝送の評価（LTspice）

```bash
python tools/generate_flyback_inrush.py   # フライバック突入電流試験（評価グラフ＋CSV）
python tools/channel_doe.py --n 500       # 高速伝送 loss/xtalk のDOE（4手法×500点）
```

- **フライバック突入電流**: 最悪位相投入時の入力突入電流を、突入電流リミッタ有無で比較
  （例: 制限なし557A → NTC 21A）。`サンプルデータ/フライバック突入電流_評価.png`
- **高速伝送DOE**: 結合損失線路の挿入損失・NEXT・FEXT を、パラメータを
  **random / Sobol / Halton / maximin** で各500点サンプリングして統計評価。
  結果は単一値に収束せず分布する。`サンプルデータ/DOE/`（各手法のCSV＋分布図）

## ファイル構成

| ファイル | 役割 |
|----------|------|
| `graph_app.py` | メイン GUI（PySide6/Qt + matplotlib 埋め込み） |

| `analysis.py` | オシロ解析（ピーク・測定・統計・FFT・THD/SNR等） |

| `advanced.py` | 高度解析（マスク/アイ/ジッタ/プロトコル解読） |
| `config_io.py` | 設定の保存／読み込み・自動復元 |
| `jp_font.py` | 日本語フォントの自動設定 |
| `tools/generate_waveforms.py` | 合成波形データ生成 |
| `tools/generate_ltspice.py` | LTspice 連携の波形データ生成 |
| `サンプルデータ/` | 動作確認用データ |
